# Development

## Build and Run

### Prerequisites

Install dependencies:

```sh
npm install
```

> **Note**
> If installation fails on Windows due to network issues, it may be caused by TLS/certificate problems. Try disabling SSL validation:
>
> ```sh
> npm config set strict-ssl false
> ```

### Build

Compile the project:

```sh
npm run build
```

### Development

Start the development server:

```sh
npm run dev
```
